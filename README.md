# Objective
The objective of this laboratory is to understand and implement fundamental and advanced computer vision techniques using Python and OpenCV, including image processing, feature extraction, segmentation, and object detection.
| Experiment No. | Title                                                  |
| -------------- | ------------------------------------------------------ |
| 1              | Image and Video Reading & Display                      |
| 2              | Image Transformations (Scaling, Translation, Rotation) |
| 3              | Histogram Analysis and Histogram Equalization          |
| 4              | Edge Detection using Canny Algorithm                   |
| 5              | Image Enhancement using Spatial Filtering              |
| 6              | Convolution, Filtering, and Motion Blur                |
| 7              | Corner Detection using Harris Algorithm                |
| 8              | Texture Segmentation using Gabor Filters               |
| 9              | Image Segmentation (Thresholding & K-Means)            |
| 10             | Object Detection (Traditional & Deep Learning)         |
